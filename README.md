# E-Assignment Management System 
A web-based platform designed for students and lecturers to manage assignment submissions efficiently. This project was developed as part of **Mini Project 2**.

## Features
- **User Authentication**: Secure login and registration for both Admins (Lecturers) and Students.
- **Role-Based Access**: 
  - **Admin**: Create assignments and view/download all student submissions.
  - **Student**: View available assignments and upload work.
- **Live Search**: Integrated AJAX-based search on the dashboard for real-time assignment filtering.
- **File Management**: Secure file upload system with unique naming conventions to prevent data loss.

## Tech Stack
- **Frontend**: HTML5, CSS3, Bootstrap 5 (Responsive Design)
- **Backend**: PHP 8.x
- **Database**: MySQL (Laragon environment)
- **Asynchronous Logic**: JavaScript Fetch API (AJAX)
